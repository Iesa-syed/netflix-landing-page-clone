# Netflix Landing Page Clone

A responsive clone of the Netflix landing page built using **HTML5** and **CSS3**. This project was created to strengthen my frontend development skills by recreating a real-world website while focusing on clean layouts, responsive design, and modern CSS techniques.

## 🚀 Features

- Responsive design for desktop, tablet, and mobile
- Modern Flexbox layout
- Hero section with background overlay
- Netflix-inspired UI
- CSS-only FAQ accordion
- Responsive navigation bar
- Custom language selector button
- Styled email signup forms
- Footer with organized links

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Google Fonts

## 📚 What I Learned

Through this project I practiced:

- Semantic HTML
- Responsive Web Design
- Flexbox layouts
- CSS positioning
- Background images and overlays
- CSS transitions
- Media queries
- Reusable styling techniques
- Building pixel-inspired UI clones
